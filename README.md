🔐 Secure File Sharing System (AES Encryption)
📌 Project Overview
This project is a secure file upload and download system that uses AES-256 encryption to protect files at rest and during transfer.

Files are encrypted before storage and decrypted only during authorized download.

🛠 Tech Stack
Python 3
Flask
PyCryptodome
HTML/CSS
Git & GitHub
🔒 Security Features
AES-256 Encryption
Secure Key Management
Encrypted File Storage
HTTPS-ready Architecture
No Plaintext File Storage
📂 Project Structure
Run pip install flask pycryptodome python app.py
